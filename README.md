# Trabalho Front VueJS

## 📖 Descrição

Este projeto foi desenvolvido utilizando **Vue.js 3** e **Vite**, com o objetivo de criar uma aplicação web para navegação e visualização de filmes.

O sistema possui funcionalidades como:

- Página inicial;
- Tela de login;
- Tela de cadastro;
- Listagem de categorias;
- Visualização de detalhes de filmes;
- Gerenciamento de favoritos;
- Navegação utilizando Vue Router.

---

## 🚀 Tecnologias Utilizadas

- Vue.js 3
- Vite
- Vue Router
- Pinia
- Axios
- JavaScript
- HTML5
- CSS3

---

## ⚙️ Pré-requisitos

Antes de executar o projeto, é necessário possuir instalado:

- Node.js (versão 18 ou superior recomendada)
- npm (instalado juntamente com o Node.js)

Para verificar as versões instaladas:

```bash
node -v
npm -v
```

---

## 📥 Instalação

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd Trabalho-FrontVUEJS
```

3. Instale as dependências:

```bash
npm install
```

---

## ▶️ Executando o Projeto

Após instalar as dependências, execute:

```bash
npm run dev
```

O Vite iniciará um servidor local e exibirá um endereço semelhante a:

```bash
http://localhost:5173
```

Abra esse endereço no navegador para utilizar a aplicação.

---

## 🏗️ Gerando Versão de Produção

Para criar a versão otimizada para produção:

```bash
npm run build
```

Os arquivos gerados ficarão na pasta:

```bash
dist/
```

---

## 👀 Visualizando a Build

Após gerar a build, é possível visualizar localmente:

```bash
npm run preview
```

## 📁 Estrutura do Projeto

```text
src/
├── assets/
├── components/
├── data/
├── router/
├── styles/
├── utils/
├── views/
├── App.vue
└── main.js
```

### Principais Diretórios

- **assets/** → imagens e arquivos estáticos.
- **components/** → componentes reutilizáveis.
- **data/** → dados utilizados pela aplicação.
- **router/** → configuração das rotas.
- **styles/** → arquivos CSS das telas.
- **utils/** → funções auxiliares.
- **views/** → páginas da aplicação.
